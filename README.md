Helectron
===
System Info Viewer

## Feature

Current:
- OS Viewer
- CPU basic Info
- Network Info

## Build

### Prerequisite
- electron
- electron-builder (if you want to package)

### Install
```
npm install
```

### Run locally
```
npm start
```
### Package
```
npm run dist
```

## Download
For binaries, please go to **release**
