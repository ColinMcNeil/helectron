===
System Info Viewer

## Feature

Current:
- OS info 
- CPU basic info

## Installing

### Prerequisite
- electron
- electron-builder

### Get Package
```
npm install
```

### Run
```
npm start
```

### Build
```
npm run dist
```

## Download
For binaries, please go to **release**
